# 🛒 Customer Shopping Behavior Analysis

## 📌 Overview
Customer Shopping Behavior Analysis is a data analytics project developed using SQL and Power BI to analyze customer purchasing patterns, sales trends, and shopping behavior. The project transforms raw retail data into meaningful business insights through data cleaning, exploratory data analysis (EDA), SQL queries, and interactive dashboard visualizations.

---

## 📂 Dataset
- **Source:** Kaggle  
- **Dataset Type:** Retail / Customer Shopping Data  
- **Rows:** 3,900+  
- **Columns:** 18  

The dataset includes customer demographics, product categories, payment methods, purchase amounts, seasonal trends, and shopping frequency.

---

## 🛠️ Tools & Technologies
- SQL  
- Power BI  
- Power Query  
- DAX  
- Excel  
- Kaggle Dataset  

---

## 📊 Project Steps
1. Imported dataset from Kaggle  
2. Cleaned and transformed the data  
3. Checked and handled null values  
4. Performed Exploratory Data Analysis (EDA)  
5. Wrote SQL queries for business insights  
6. Created DAX measures in Power BI  
7. Designed an interactive dashboard  

---

## 📈 Dashboard Features
- KPI Cards  
- Sales Analysis  
- Customer Demographics  
- Product Category Insights  
- Seasonal Trends  
- Payment Method Analysis  
- Interactive Filters & Slicers  

---

## 🔍 Key Results
- Clothing and electronics generated the highest sales  
- Female customers contributed more to purchases  
- Winter season showed higher sales activity  
- Digital payment methods were most preferred  
- Discounts positively influenced customer purchases  

---

## ▶️ How to Run
1. Download the dataset from Kaggle

3. Perform data cleaning and analysis  
4. Open the `.pbix` file in Power BI Desktop  
5. Explore the interactive dashboard
